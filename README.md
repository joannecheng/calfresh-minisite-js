# CalFresh Minisite
<br>
## Install dependencies

```
npm install
```


## Develop locally with webpack-dev-server
1. Run

```
npm run start
```

2. In your browser, navigate to: [http://localhost:8080/](http://localhost:8080/)
## For bundled output

```
npm run build
```

## For production-ready output

```
npm run build:prod
```
